RESPONSIVE WEB DESIGN

RESUMEN

Dentro de este artículo se describe brevemente los antecedentes, funciones y beneficios de la técnica del Diseño Web Responsivo (Responsive Web Design). Cabe destacar la técnica del Responsive Web Design consiste en la adaptabilidad de las páginas web a todo tipo de medios, pantallas y dispositivos. Es decir, este soluciona el problema de la visibilidad en la gama de dispositivos móviles, permite un mejor posicionamiento del sitio en los buscadores y crea las bases para incorporar nuevos servicios como el Comercio Electrónico. Palabras clave: Diseño Web Responsivo, adaptabilidad, dispositivos, posicionamiento, técnica.

INTRODUCCIÓN

El crecimiento y expansión de los sistemas móviles, tabletas y otros dispositivos ha afectado a campos diferentes en las últimas décadas. Tal es la producción de sitios de internet que buscan métodos innovadores para presentar contenidos con soluciones efectivas en dispositivos móviles, teniendo en cuenta su versatilidad y diferentes formas de comunicación. Cabe destacar que la posibilidad de navegar por Internet, a través de los teléfonos inteligentes o Smartphones revolucionó el diseño Web a partir del año 2007. Sin lugar a dudas las posibilidades y ventajas de este método han sido aceptadas por la gran mayoría de los desarrolladores de páginas web.

DESARROLLO

Claramente la técnica Responsive Web Design nos permite crear sitios adaptables a las condiciones del ordenador o dispositivo desde donde se van a acceder, sobre todo en lo que tiene relación con la pantalla del sistema donde se están visualizando. Del mismo modo, el diseño receptivo puede manejar la navegación web tanto de escritorio como móvil a través de tres componentes técnicos: las rejillas de fluidos, las imágenes flexibles y las consultas de medios de comunicación. Ahora bien, antes existían solo ordenadores donde se consumían los contenidos y sin embargo hoy podemos consultar Internet en cualquier tipo de dispositivo. Móviles, tablets, televisores, libros electrónicos, relojes e incluso neveras permiten acceder a la Web y sus servicios. Son múltiples medios con distintas características y los diseñadores web deben preocuparse porque los sitios se vean bien en cualquiera de ellos. Cabe destacar que el RWD se hace posible gracias a la introducción de las Media Queries en las propiedades de los estilos CSS en su versión número 3. Esta incluye consultas más específicas basadas en las características de los dispositivos como el ancho, alto y capacidad de colores. Es decir, el Responsive Web Design trabaja con porcentajes para establecer el ancho de cada elemento del layout, lo que facilita que los contenedores sean más adaptables, fluidos y no dependan de un ancho fijo, además se ahorra el despliegue de barras deslizadoras. Cabe destacar los beneficios de tener un sitio web responsive: Mejora el branding corporativo. Se reduce en gran medida el tiempo de desarrollo de la web. Proporciona una mayor viralidad de los contenidos y las imágenes. El tiempo de carga se reduce exponencialmente. Se evitan los contenidos duplicados. Por lo tanto, al redimensionarse y colocarse los elementos de la web, se permite que se visualicen de forma correcta en las diferentes pantallas que se utilicen, lo que provoca una buena experiencia de usuario ya que se le facilita el trabajo de búsqueda e información.

CONCLUSIONES, RECOMENDACIONES, TRABAJOS FUTUROS

Con la aparición de nuevos sistemas y dispositivos únicos, surgen nuevas necesidades que se satisfacen con tecnologías rediseñadas, utilizando tecnologías existentes o con nuevas tecnologías y estándares. Aunque para este campo en crecimiento, el diseño web ofrece soluciones que abren un panorama de investigación tecnológica muy importante para desarrolladores y sistemas portátiles. A medida que evolucionan los estándares HTML y CSS, se crean elementos reutilizables para minimizar el tiempo de desarrollo y admitir navegadores y dispositivos actuales. Es sumamente importante utilizar la técnica Responsive Web Design en los sitios web. Ya gran parte de las búsquedas en Internet hoy en día se realizan en un teléfono o tableta, y si la página no es responsive y su presentación se ve mal, al usuario simplemente no le llamara la atención. Por tanto, el diseño receptivo está cobrando impulso como una tendencia de desarrollo web que se está convirtiendo en la web del futuro. En un corto período de tiempo, la mayoría de los sitios web se adaptarán para que sean receptivos. Este tipo de diseño web se ha vuelto muy popular y se usa en Internet por tres razones muy simples: es barato, es rápido y funciona. Destacando que Google recomienda un sitio web receptivo en lugar de crear un sitio web móvil desde cero para optimizar la experiencia del usuario, evitar contenido duplicado y también brindar una apariencia uniforme.
